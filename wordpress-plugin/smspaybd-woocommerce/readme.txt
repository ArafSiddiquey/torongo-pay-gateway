Torongo Pay WooCommerce Gateway redirects customers to your self-hosted Torongo Pay payment page and marks WooCommerce orders paid after SMS verification.

== Setup ==
1. Upload the smspaybd-woocommerce folder to wp-content/plugins/.
2. Activate the plugin.
3. Go to WooCommerce > Settings > Payments > Torongo Pay.
4. Enter your gateway URL, for example https://pay.yourdomain.com.
5. Enter the same webhook secret from your Torongo Pay admin Gateway Setup.
6. Enable the gateway and save.
