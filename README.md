# Torongo Pay Semi-Auto Gateway

SMS-based semi-auto payment gateway built with Laravel + MySQL and an Android SMS Reader app.

## What is included
- Customer payment checkout UI
- bKash/Nagad/Rocket method flow
- bKash Send Money / Payment option modal
- QR auto show when uploaded
- 15-minute countdown
- Auto SMS verification
- Manual TrxID verification
- Dark admin dashboard
- SMS devices and API key management
- Google Sheet backup webhook
- Bangla/English editable text
- Android SMS Reader source
- Standalone interactive HTML preview
